# Personal Portfolio Website

Welcome to my **Personal Portfolio Website**! This repository contains the source code for Task 3 of the **CodeAlpha Frontend Development Internship**.

## Overview
A professional, fully responsive, and dark-themed personal portfolio website. It is designed to showcase my journey as an IT graduate, highlighting my skills, projects, internships, and academic achievements.

## Features
- **Modern Dark Theme:** An elegant UI with gold and purple accent colors.
- **Sticky Navigation:** Smooth scrolling navigation for a seamless user experience.
- **Responsive Design:** Optimized for desktops, tablets, and smartphones using CSS Grid/Flexbox.
- **Sections:** Detailed sections for About Me, Skills, Projects, Experience, Rewards, and Contact.
- **Direct Resume Download:** Easy access to my latest PDF resume.

## Tech Stack
- **HTML5:** Semantic structure and accessible web elements.
- **CSS3:** Advanced styling, responsive layouts, and smooth transition animations.
- **JavaScript** Implemented for smooth scrolling behavior.
